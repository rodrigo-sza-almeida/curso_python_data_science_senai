# Notebooks – Curso Programação em Python para Data Science (SENAI)

Este repositório reúne os notebooks desenvolvidos durante o curso **"Programação em Python para Data Science"** ministrado pelo **SENAI - "Almeirante Tamandaré"**.  
O objetivo é registrar minha evolução na linguagem Python aplicada a dados, além de servir como base para estudos futuros e para compartilhamento com a comunidade.

---

## 🧠 Conteúdo do curso

Ao longo do curso, trabalhei os seguintes temas principais:

- Fundamentos da linguagem **Python**
- Estruturas de dados (listas, dicionários, tuplas, conjuntos)
- Estruturas de controle (condicionais, laços, funções)
- Introdução a notebooks (Jupyter/Google Colab)
- Manipulação de dados com **pandas**
- Operações numéricas com **numpy**
- Visualização de dados com **matplotlib** e/ou **seaborn**
- Limpeza, transformação e preparação de dados
- Análise Exploratória de Dados (EDA)
- Boas práticas de codificação e organização de notebooks

---

## 📂 Estrutura dos notebooks

Abaixo, uma visão geral dos notebooks disponíveis neste repositório:

- `aula_tipos_dados.ipynb`
  `aula_variaveis.ipynb`  
  Introdução à linguagem, tipos de dados, operações básicas e primeiros scripts.

- `aula_condicionais.ipynb`
  `aula_for.ipynb`
  `aula_while.ipynb`
  `aula_listas.ipynb`
  `aula_tuplas`
  `aula_dicionarios`  
  Trabalho com listas, dicionários, tuplas, estruturas condicionais e laços.

- `aula_funcoes.ipynb`
  `aula_expressoes_lambda.ipynb`  
  Criação de funções, organização de código e reutilização de lógica.

- `aula_pandas.ipynb`  
  Leitura de arquivos, criação de DataFrames, seleção e filtragem de dados.

- `aula_pandas_excel.ipynb`  
  Tratamento de valores ausentes, correção de tipos, padronização de campos.

- `06_analise_vg_sales.ipynb`  
  Estatísticas descritivas, gráficos e exploração de padrões nos dados.

- `projeto_final_rodrigo.almeida.ipynb`  
  Notebook que integra os principais conceitos do curso em um projeto aplicado.

## ▶️ Como executar os notebooks

1. Faça o clone deste repositório:
   ```bash
   git clone https://github.com/rodrigo-sza-almeida/curso_python_data_science_senai.git

2. Instale as dependências necessárias (se estiver usando ambiente local com pip):
   pip install -r requirements.txt

3. Abra os notebooks:
   - Usando Jupyter Notebook:
     jupyter notebook

   - Ou carregando os arquivos diretamente no Google Colab.
